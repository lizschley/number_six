Python decorators(not decorator design patterns) add functionality to functions and methods at definition time.

For example, when you write @staticmethod over a method in Python. GeekForGeeks (generally an excellent resource) provides an excellent definition of decorators. Basically decorators are used to add extra functionality to a function without changing the functionality itself. For example, before the function you could check the begin time and afterwards check the end time.

Here is an example showing how to send in variables and how to return values. The readthedocs site has a great example showing the order of when the wrapper and functions are called. The following code snippet combines the examples, to show both the call order and how to return values from the function that is being decorated.

def decorator_function(func):
    print('#1 - inside decorator')
    def wrapped_f(*args, **kwargs):
        print('#4 & 9 - Inside wrapped_f()')
        print('#5 & 10 - Decorator arguments:', args, kwargs)
        returned_value = func(*args)
        print('#7 & 12 - After f(*args)')
        return returned_value
return wrapped_f

@decorator_function 
def show_arguments(a1, a2, a3, a4):
    print('#6 & 11 - show_arguments arguments:', a1, a2, a3, a4)
    return 'done printing'

print('#2 - After decoration')
print('#3 Preparing to call show_arguments()')
ret = show_arguments('say', 'hello', 'argument', 'list')
print(f'#8 - after first show_arguments() call, returned_value == {ret}')
show_arguments('a', 'different', 'set of', 'arguments')
print('#13 - after second show_arguments() call')
This puts out the following data:

#2 - After decoration
#3 Preparing to call show_arguments()
#4 & 9 - Inside wrapped_f()
#5 & 10 - Decorator arguments: ('say', 'hello', 'argument', 'list') {}
#6 & 11 - show_arguments arguments: say hello argument list
#7 & 12 - After f(*args)
#8 - after first show_arguments() call, returned_value == done printing
#4 & 9 - Inside wrapped_f()
#5 & 10 - Decorator arguments: ('a', 'different', 'set of', 'arguments') {}
#6 & 11 - show_arguments arguments: a different set of arguments
#7 & 12 - After f(*args) #13 - after second show_arguments() call
